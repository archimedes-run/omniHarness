# Security Policy

## Supported Versions

As omni-harness doesn't provide an official release yet, please use the latest version for the security updates.
Currently, we have two branches to maintain:
* main branch for omni-harness 2.x
* main-1.x branch for omni-harness 1.x 

## Reporting a Vulnerability

Please go to https://github.com/archimedes-run/omni-harness/security to report the vulnerability you find.
